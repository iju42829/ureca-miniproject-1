# Ureca Mini Project 1 - 휴대폰 주문 시스템

## 프로젝트 개요
이 프로젝트는 콘솔 기반 **휴대폰 주문 시스템**으로,
- **회원**은 휴대폰 주문 및 주문 기록 확인이 가능하며,
- **관리자**는 휴대폰 등록, 재고 수정, 판매 중지 등의 관리를 수행할 수 있습니다.

## 주요 기능
### 회원 기능
- 회원 가입 및 로그인
- 비밀번호 변경
- 주문 내역 확인

### 휴대폰 관리
- 휴대폰 등록 및 수정
- 재고 관리 및 판매 중지 처리
- 휴대폰 목록 조회

### 주문 기능
- 휴대폰 주문 생성
- 주문 기록 확인

## 기술 스택
- **Java**
- **MySQL**
